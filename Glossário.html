Principais tags HTML e seus significados.

As tags na estrutura HTML servem para organizar, definir e sistematizar a construção
da página. Algumas delas são inclusive obrigatórias para se configurar um documento
HTML, enquanto outras são opcionais, porém o seu uso representa uma atitude de boas
práticas. Abaixo listei as principais, veja:

<p>
A tag p se utiliza quando queremos agrupar informações através de um parágrafo.
<a>
Utilizada como elemento âncora, para navegação entre páginas web, e-mails e outros.
<button>
O elemento button refere-se a um botão ciclável na página web.
<footer>
É o utilizado para inserir o rodapé da página, contendo geralmente informações sobre o autor.
